# MyFirstGainPlugin
My first JUCE audio plugin. 

![The Meanest Plugin](res\reaper_azDnEEYsyt.png)

# Build Environment
* Visual Studio 2019

# Build Steps
1. Download JUCE.
2. Run Projucer.
3. Set path to JUCE modules.
4. Set target plugin format.
5. Create project.
6. Compile in Release mode.
7. Use an IDE to scan your build directory.
8. Have fun.
